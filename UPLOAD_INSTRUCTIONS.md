### **STEP 2: Create New Repository**

1. Click **"+"** icon (top right)
2. Select **"New repository"**
3. Fill in:
   - **Repository name:** `treatall-website`
   - **Description:** "TreatAll Professional Elderly Care Website"
   - **Public** (MUST be public for free hosting)
   - **DO NOT** check "Add a README file"
4. Click **"Create repository"**

---

### **STEP 3: Upload Files**

#### **Method A: Drag & Drop (EASIEST)**

1. You'll see a quick setup page
2. Click **"uploading an existing file"** link
3. **Drag all 4 files** from your folder into the browser
   - OR click **"choose your files"** and select all 4 files
4. All files will upload at once
5. Scroll down
6. Click **"Commit changes"** (green button)

‚úÖ **All files uploaded!**

#### **Method B: Upload One by One**

1. Click **"Add file"** ‚Üí **"Upload files"**
2. Upload **index.html** first
3. Click **"Commit changes"**
4. Repeat for each file:
   - robots.txt
   - sitemap.xml
   - README.md

---

### **STEP 4: Enable GitHub Pages**

1. Click **"Settings"** tab (at top)
2. Scroll down left sidebar
3. Click **"Pages"** (under "Code and automation")
4. Under **"Source"**:
   - Select **"Deploy from a branch"**
5. Under **"Branch"**:
   - Select **"main"** (or "master" if that's shown)
   - Keep **"/ (root)"**
6. Click **"Save"**
7. **Wait 2-3 minutes**
8. **Refresh the page**
9. You'll see: **"Your site is live at https://..."**

‚úÖ **WEBSITE IS LIVE!**

## üîß IF SOMETHING GOES WRONG

### **Files won't upload?**
- Make sure you're on the repository page
- Try uploading one file at a time
- Check internet connection
- Try different browser

### **"404 - File not found" error?**
- Make sure file is named **exactly** `index.html` (lowercase)
- Wait 3-5 minutes after enabling Pages
- Hard refresh: Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)

### **Website not showing?**
- Wait 5 minutes and try again
- Make sure GitHub Pages is enabled (Settings ‚Üí Pages)
- Check the branch is set to "main"
- Try in private/incognito browser window

### **Changes not appearing?**
- Wait 1-2 minutes after uploading
- Clear browser cache
- Try Ctrl+Shift+R for hard refresh
